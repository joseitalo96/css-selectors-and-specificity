CSS selectors and specificity

CSS selectors

SELECTOR MEANING EXAMPLE
UNIVERSAL SELECTOR Applies to all the elements in the document * {}
Targets all the elements on the page
TYPE SELECTOR Matches element names h1, h2, h3 {}
Targets all the elements h1, h2, h3
CLASS SELECTOR Applies to all the elements that have a class with the same value as the one specified after the dot symbol .code {}
Targets the elements that has a class of code
p.note {}
Targets the paragraph elements whose class has a value of note
ID SELECTOR Applies to all the elements that has an ID attribute with the same value as the one specified after the pound symbol #computer {}
Targets the elements that has an ID of computer
